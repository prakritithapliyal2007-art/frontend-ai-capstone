# Frontend AI Capstone

## Overview
This repository contains my Frontend AI Engineering capstone project.

## Getting Started

### Prerequisites
- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (LTS recommended; needed when the React app is added)

### Setup
```bash
git clone https://github.com/prakritithapliyal2007-art/frontend-ai-capstone.git
cd frontend-ai-capstone
```

The frontend app scaffold is not in place yet. After it is added, install dependencies and start the dev server with:

```bash
npm install
npm run dev
```
